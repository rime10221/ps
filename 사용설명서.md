# E-daily 입원/응급 자동화 프로그램 — 사용설명서

E-daily의 **02 ADMISSION/DISCHARGE** 와 **03 EMERGENCY ROOM** 작성 화면을
엑셀/텍스트 입력만으로 자동으로 채워 넣는 프로그램입니다.
저장(제출)은 자동으로 하지 않으며, 반드시 사용자가 확인 후 직접 누릅니다.

---

## 0. 준비물

- **Google Chrome** 설치 (필수)
- **edaily_auto.exe** (이 프로그램)
- EMR에서 추출한 입력 파일 (아래 1번 참고)

> 💡 `입원목록.xls` 와 `ER.xls` 를 **exe 와 같은 폴더**에 두면 경로가 자동으로 입력됩니다.
> (다른 위치에 있으면 프로그램에서 [찾아보기]로 선택하면 됩니다.)

---

## 1. 입력 파일 만들기 (EMR에서 추출)

### 1-1. 입원목록.xls  — 입원환자 명단
1. EMR에서 **원무관리 → 예약정보 → 입원예약조회** 이동
2. **[출력]** 버튼 클릭  ⚠️ *(엑셀 버튼이 아니라 '출력' 버튼입니다)*
3. 출력 미리보기 창에서 **엑셀 아이콘** 클릭
4. 열린 엑셀 창에서 **[다른 이름으로 저장]**
   - 파일 이름: `입원목록.xls`
   - 파일 형식: **Excel 97-2003 통합 문서 (*.xls)**  ⚠️ *(`.xlsx` 로 저장하면 프로그램이 못 읽습니다)*
5. exe 와 같은 폴더에 저장

### 1-2. 진단명  — 환자 1명의 진단 목록 (파일 저장 불필요, 복사만)
1. 해당 환자 **처방창** 열기
2. 오른쪽 위 **[진단]** 버튼 클릭
3. 진단명 목록에서 **맨 첫 행의 진단명을 클릭**
4. **Shift** 를 누른 채 **맨 아래 행까지** 선택 (전체 긁기)
5. **Ctrl + C** 로 복사
6. → 프로그램 ② 진단명 칸에 **Ctrl + V** 로 붙여넣기 (3-2 참고)

### 1-3. ER.xls  — 응급실 환자 명단
1. EMR에서 **진료대상리스트 → 응급** 이동
2. **[출력]** 버튼 클릭
3. **엑셀 아이콘** 클릭 → **[다른 이름으로 저장]**
   - 파일 이름: `ER.xls`
   - 파일 형식: **Excel 97-2003 통합 문서 (*.xls)**
4. exe 와 같은 폴더에 저장

> (진단명 복사 방식은 1-2 와 동일, ER 은 입원목록과 동일한 출력→엑셀 저장 방식입니다.)

---

## 2. 프로그램 실행 & 크롬 준비

1. **edaily_auto.exe** 더블클릭 → 프로그램 창이 열립니다.
2. 프로그램 맨 위 **⓪ [크롬 디버깅 모드로 열기]** 버튼 클릭
   - E-daily 사이트가 열리는 **전용 크롬 창**이 뜹니다.
   - ⚠️ 이 크롬 창으로 작업해야 프로그램이 연결됩니다. (평소 쓰는 크롬과 별개)
3. 그 크롬에서 **E-daily 로그인**
4. 작성할 페이지를 띄웁니다:
   - 입원/진단명 작업 → **02 ADMISSION/DISCHARGE** 페이지
   - 응급실 작업 → **03 페이지의 EMERGENCY ROOM**

---

## 3. 단계별 사용법

### ① 기본정보 입력 (입원환자)
1. 크롬에서 **02 ADMISSION** 페이지가 열려 있는지 확인
2. 프로그램 **① [환자 불러오기 & 기본정보 폼 채우기]** 클릭
3. `입원목록.xls` 를 읽어 **입원 명수**를 설정하고, 환자별로
   등록번호 / 병실 / 이름 / 성별 / 나이 / 보험·비보험 / 전문의 를 자동 입력합니다.
4. 아래 **[대상 환자]** 드롭다운에 환자 목록이 채워집니다.

### ② 진단명 입력 (환자 1명씩)
1. **[대상 환자]** 드롭다운에서 진단명을 넣을 **환자 1명 선택**
2. 큰 입력칸에 1-2 에서 복사한 진단명을 **Ctrl + V** 로 붙여넣기
3. **② [선택 환자에 진단명 입력]** 클릭
   - 진단명(영문 상병명)으로 코드를 자동 조회·매칭해 진단코드를 입력합니다.
   - 상병이 여러 개면 '추가'까지 자동으로 반복하고, 각 행의 보험/전문의도 함께 채웁니다.
4. 다른 환자도 진단명을 넣으려면 **①은 다시 안 해도 되고**, ②에서 환자만 바꿔가며 반복하면 됩니다.

> ✅ 진단명↔환자 연결은 프로그램이 자동으로 맞추지 않습니다.
> **어떤 환자에게 어떤 진단을 넣을지는 사용자가 EMR에서 직접 확인**하고 대상 환자를 선택하세요.

### ③ EMERGENCY ROOM 입력 (응급실)
1. 크롬에서 **03 페이지 (EMERGENCY ROOM)** 가 열려 있는지 확인
2. `ER.xls` 경로 확인 (자동 입력, 아니면 [찾아보기])
3. **③ [응급실(EMERGENCY ROOM) 환자 채우기]** 클릭
   - ER 명수를 설정하고, 등록번호 / 이름 / 성별 / 나이 를 채우며
   - **병실은 모두 `ER`**, **입퇴원여부는 모두 `퇴원`** 으로 고정 입력합니다.

### 저장
- 프로그램은 **저장을 자동으로 누르지 않습니다.**
- 브라우저에서 입력 내용을 **눈으로 확인**한 뒤, E-daily 화면의 **[저장]** 또는 **[저장 후 다음]** 을 직접 누르세요.

---

## 4. 주의사항

- 입력 후 항상 **브라우저에서 내용을 검토**하고 저장하세요. (자동 저장 안 함)
- 엑셀은 반드시 **.xls (97-2003)** 형식으로 저장 (`.xlsx` 불가)
- **⓪ 버튼으로 연 크롬**에서만 동작합니다. 그 창을 닫으면 다시 ⓪부터 하세요.
- 진단명 매칭은 영문 상병명 기준입니다. 드물게 못 찾는 상병은 로그창에 `매칭 실패`로 표시되니, 해당 항목만 수동으로 입력하세요.
- 환자 정보가 담긴 xls/txt 파일은 외부(메일·클라우드·GitHub 등)로 공유하지 마세요.

## 5. 문제 해결 (FAQ)

| 증상 | 해결 |
|---|---|
| "페이지 탭을 찾지 못했습니다" | ⓪ 크롬에서 해당 페이지(02/03)를 먼저 열어두세요. |
| xls 를 못 읽음 | `.xlsx` 로 저장했는지 확인 → **.xls (97-2003)** 로 다시 저장 |
| 진단코드 일부 `매칭 실패` | 해당 상병만 코드조회에서 수동 입력 |
| 크롬이 안 열림 | Chrome 설치 여부 확인 |
| 명수/행이 안 맞음 | 페이지를 새로고침한 뒤 ①(또는 ③)부터 다시 실행 |
| 프로그램이 크롬에 연결 안 됨 | ⓪ 버튼으로 연 크롬을 닫지 않았는지 확인, 닫았으면 ⓪부터 다시 |

## 진행 순서 요약

```
[EMR] 입원목록.xls / ER.xls 저장, 진단명 복사
      │
[exe] ⓪ 크롬 열기 → 로그인 → 02/03 페이지 띄우기
      │
      ① 기본정보 채우기 (입원목록.xls)
      ② 환자 선택 + 진단명 붙여넣기 → 진단코드 입력   (환자별 반복)
      ③ 응급실 채우기 (ER.xls)
      │
[크롬] 내용 확인 → 저장 직접 클릭
```
